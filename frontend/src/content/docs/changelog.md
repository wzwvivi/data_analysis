# 版本变更记录

本页维护平台对外的版本发布记录。发布日期以实际部署为准；文档中心内容与当前运行版本保持一致。

## v1.0.1 — 术语收敛（专项分析）

### 优化

- 统一一级分组术语：`事件分析` 调整为 `专项分析`，二级保留飞管事件分析、飞控事件分析、自动飞行性能分析与 TSN 异常检查。
- 前端路由兼容：历史路径 `/event-analysis` 自动重定向到 `/fms-event-analysis`，避免旧书签失效。
- 后端 API 进入兼容期：`/api/event-analysis/*` 保持可用，并标记为即将下线（建议改用 `/api/fms-event-analysis/*`）。

## v1.0.0 — 正式发布版

> 内部首个正式发布版本，作为后续迭代的稳定基线。

### 新增

- 上线公开 **产品着陆页**，面向未登录用户介绍平台定位与核心能力。
- 上线站内 **帮助中心**：分组导航、全文搜索、页内目录、深链接锚点、打印样式一体化。
- 提供完整的 **模块文档体系**，覆盖上传解析、任务中心、事件分析、异常检查、工作台、协议与网络配置管理、系统管理等全部模块。
- 顶部导航栏提供 `?` 快捷按钮，任意页面均可直达帮助中心。

### 优化

- 按 **服务器容量** 调整解析并发与资源策略，保障单任务解析体验。
- **视频共享数据** 处理链路引入串行化与快速 remux 路径，降低高峰期资源争抢。
- 侧边栏结构按模块分组统一；用户、构型、共享数据等管理员入口收敛至 `系统配置`。
- 文档文本整体按发布版语言重写，统一术语与表达。

### 运行与治理

- 历史任务结果 **长期保留**，便于跨版本复盘。
- 权限模型按页面维度统一；管理员角色默认拥有全部页面权限。
- 文档中心中文为主，必要英文术语保留以方便对照。
