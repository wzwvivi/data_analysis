# 平台共享数据（管理员）

平台共享数据用于管理跨任务可复用的数据资产，包括共享文件及其处理状态。

## 适用角色

仅管理员可进行新增、更新、删除等操作。普通用户通常不具备该页面权限。

## step-1 上传共享数据

1. 进入系统配置下的 `平台共享数据`。
2. 上传文件并填写必要元数据。
3. 系统自动记录创建时间与处理状态。

## step-2 跟踪处理状态

1. 对视频类文件，平台可能执行浏览器兼容预处理。
2. 在列表中查看状态（待处理、处理中、成功、失败）。
3. 失败时可查看原因并重新上传。

## step-3 清理与治理

1. 对失效资产进行下线或删除。
2. 定期审查存储占用和可用性。

## 常见问题

### 上传后不可预览

- 可能仍在转码队列中，稍候刷新状态。
- 若持续失败，检查原始编码格式与日志信息。
