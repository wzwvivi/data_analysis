# 架次比对

架次比对位于 `试验工作台 → 对比所选架次`，用于**跨架次**汇总关键飞行指标、事件数量与异常信息，面向复盘场景。

## 与 TSN 异常检查的区别

- **架次比对（本页）**：对象是两个及以上**架次**，每个架次下需选一个解析任务；指标偏向飞行剖面、姿态、事件数量等复盘维度。
- **TSN 异常检查**：对象是两个**解析任务**本身，检查时间同步、端口覆盖、周期连续性与抖动等数据包级问题。入口在 `专项分析 → TSN 异常检查`。

遇到“这两次飞行差在哪”时用架次比对；遇到“这两份抓包谁丢了包、谁周期抖动”时用 TSN 异常检查。

## 典型场景

- 对比同机型不同架次在飞行剖面或姿态上的表现差异。
- 验证某次改动前后关键指标的改善情况。
- 为评审沉淀可解释的横向证据。

## step-1 选择参与对比的架次

1. 在试验工作台勾选至少 2 个架次，点击「对比所选架次」。
2. 页面支持通过顶部下拉再增删架次；已选架次会以 Tag 列出。

## step-2 为每个架次选择解析任务

1. 每个架次下会列出该架次匹配到的解析任务（含文件名与状态）。
2. 默认选中最近的一条；可根据需要切换到其他解析任务。
3. 所有架次都已指定解析任务后，点击「加载/刷新对比」。

## step-3 查看对比结果

- **指标对比表**：按指标列出每个架次的值；点击列头的「打开架次」可跳到该架次工作台做进一步下钻。
- **事件数量对比柱图**：在同一视图下看 FMS/FCC/自动飞行/比对异常的数量差异。

## 输入与输出

- 输入：2+ 架次、每个架次对应一个解析任务。
- 输出：指标差异表 + 事件数量柱图；支持单架次下钻到工作台。

## 常见问题

### 某个架次无候选解析任务

- 确认该架次下已有已完成的解析任务，且任务与当前账号可见的协议版本匹配。

### 两个任务看起来都没差别

- 检查是否选到了相同解析任务；也可回到任务中心确认任务参数是否一致。

### 我想做“两份抓包”的差异检查

- 请切换到 `专项分析 → TSN 异常检查`。
