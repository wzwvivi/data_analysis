# 任务中心

任务中心用于统一管理解析任务和后续分析入口，是日常排查的核心工作台之一。

## 模块价值

- 实时查看任务状态和执行结果。
- 快速进入解析结果、事件分析页面。
- 集中查看失败任务和错误信息，提升排障效率。

## step-1 查看任务列表

1. 进入任务中心，按时间和状态筛选。
2. 关注运行中、失败任务优先级。
3. 按任务名称或编号定位目标任务。

## step-2 打开任务详情

1. 点击任务进入详情页。
2. 查看输入参数、执行状态、结果摘要。
3. 通过快捷入口跳转到结果分析页面。

## step-3 失败任务处理

1. 阅读错误提示和上下文参数。
2. 回到上传解析页面调整参数并重试。
3. 对重复失败问题记录到 FAQ，减少重复排查。

## 输入与输出

- 输入：筛选条件、任务标识。
- 输出：任务状态、错误信息、结果页面链接。

## 常见问题

### 成功任务看不到结果

- 先刷新页面确认结果索引已写入。
- 若持续为空，检查后台结果目录是否可读。

### 任务状态长时间不变化

- 可能队列繁忙或后台处理阻塞，建议联系管理员排查进程负载。
