# 常见问题

本页汇总平台发布后的高频问题与排查建议。遇到问题请优先按本页提示定位，问题仍未解决时再联系平台管理员。

## 上传与任务

### Q1：上传时页面提示失败，怎么办？

- 优先核对文件大小是否超过单次上传上限（当前策略为 2GB），以及网络是否稳定。
- 若服务器剩余磁盘空间低于保护阈值，平台会主动拒绝上传，请联系管理员清理或扩容。
- 使用较大文件时建议在网络环境稳定的机器上提交。

### Q2：任务长时间处于 `排队中` 或 `运行中`

- 平台对解析并发有保护策略，高峰期任务会按序执行。
- 若数小时内仍无变化，请联系管理员查看后台负载及任务日志。

### Q3：任务失败如何定位原因？

1. 打开任务详情页查看错误信息和提交参数。
2. 核对原始文件格式、协议版本、端口范围是否匹配。
3. 必要时调整参数后重新提交；反复失败建议记录到团队 FAQ。

## 权限与账号

### Q4：登录后看不到某个模块入口

当前账号未开通该页面权限。请联系管理员在 `用户管理` 中分配权限；管理员角色默认拥有全部页面权限。

### Q5：端口下拉为空

- 可能是账号的端口可见范围未配置。
- 也可能是所选协议版本尚未加载完成，请稍候重试。

### Q6：如何修改密码

在任意页面点击右上角用户头像 → `修改密码`，按照提示完成修改。建议首次登录后立刻修改初始密码。

## 配置与协议

### Q7：上传页面看不到可选协议版本

- 请确认目标版本已发布并处于 `Available` 状态。
- 请确认账号具备访问 `设备协议管理` 或对应模块的权限。
- 若版本新建不久，建议刷新页面后重试。

### Q8：发布新版本后，历史任务会被影响吗？

不会。历史任务始终绑定其提交时使用的协议或配置版本；新版本仅对后续新提交的任务生效。

## 视频与共享数据

### Q9：上传视频后无法立刻预览

视频在后台会进行浏览器兼容处理（转码或容器转换）。处理期间状态保持 `处理中`，通常几分钟内可完成。若长时间未成功，请查看任务错误信息。

### Q10：飞行助手入口为何有时不显示？

该入口需要满足两个条件：
1. 管理员在公开配置中填写了飞行助手地址。
2. 当前用户具备 `flight-assistant` 页面权限。

任一条件不满足，入口都会被隐藏。

## 文档与使用

### Q11：如何快速定位到某个操作步骤？

- 在帮助中心顶部搜索关键词。
- 使用右侧 `本页目录` 或 `#step-x` 锚点直达步骤。
- 常用文档可直接收藏，URL 形如 `/help/upload#step-3`。

### Q12：文档是否可以打印或导出？

可以。帮助中心每篇文档右上角均有 `打印` 按钮，会使用专用打印样式，便于导出 PDF 或纸质分发。

## 发布前自检建议

1. 主链路走通：上传 → 任务中心 → 结果分析。
2. 关键页面权限按角色分配准确。
3. 构型与协议的变更已记录、已审批、已发布。
4. 版本变更记录已更新到发布说明。
