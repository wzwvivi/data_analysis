# 飞管事件分析

飞管事件分析面向飞行管理系统相关事件的提取、筛选与定位，适用于功能验证与问题复盘。

## 典型场景

- 验证飞管逻辑是否按预期触发。
- 排查特定时段的事件链路。
- 输出任务级事件摘要用于评审材料。

## step-1 进入分析任务

1. 从任务中心或结果页进入飞管事件分析。
2. 确认任务上下文与时间范围。

## step-2 筛选与定位

1. 按事件类型、严重级、时间区间筛选。
2. 查看事件明细与上下文字段。
3. 与工作台数据交叉验证结论。

## step-3 结果沉淀

1. 记录关键事件及触发条件。
2. 将发现的问题转入缺陷流或测试回归清单。

## 输入与输出

- 输入：解析任务结果、筛选条件。
- 输出：结构化事件列表、诊断结论。

## 常见问题

### 事件数量异常偏少

- 检查任务是否完整解析成功。
- 检查筛选条件是否过于严格。
