# 自动飞行性能分析

自动飞行性能分析用于评估自动飞行阶段的核心指标表现，支持任务级和趋势级观察。

## 典型场景

- 评估自动阶段控制稳定性。
- 对比不同版本在自动阶段的性能表现。
- 为发布评审提供指标证据。

## step-1 选择任务

1. 进入自动飞行性能分析模块。
2. 选择目标任务或历史分析任务。

## step-2 查看指标与图表

1. 浏览核心指标卡片。
2. 结合图表观察关键区间变化。
3. 记录异常时刻并回链至原始任务。

## step-3 复盘与导出

1. 汇总指标异常与结论。
2. 用于发布评审或回归基线对比。

## 输入与输出

- 输入：解析结果、自动飞行阶段数据。
- 输出：指标统计、趋势图、任务级评估结论。

## 常见问题

### 图表显示为空

- 检查任务是否包含自动飞行阶段数据。
- 检查筛选条件是否导致无数据。
