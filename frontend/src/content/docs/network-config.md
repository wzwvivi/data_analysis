# TSN 网络配置

TSN 网络配置模块负责网络侧配置资产的版本化管理，支持草稿编辑、审批流和版本查看。

## 模块价值

- 避免口头配置导致的解析口径漂移。
- 通过审批流提升变更可追溯性。
- 通过版本管理支撑历史任务复盘。

## step-1 创建或编辑草稿

1. 在配置列表新建草稿。
2. 按模板填写字段并保存。
3. 使用草稿编辑器校验结构与语义。

## step-2 发起变更请求

1. 在草稿页发起变更请求。
2. 填写变更说明与影响范围。
3. 等待审批并跟踪状态。

## step-3 查看发布版本

1. 在版本列表查看已发布版本。
2. 打开版本详情核对关键字段。
3. 在上传解析时选择匹配版本。

## 输入与输出

- 输入：网络配置草稿、审批信息。
- 输出：可发布的版本快照和审计记录。

## 常见问题

### 变更请求无法提交

- 检查草稿是否通过基础校验。
- 检查必填字段和审批说明是否完整。

### 历史版本如何回看

- 在版本详情页可查看完整快照与变更上下文。
